# 应用管理模块产品文档

## 1. 模块目标
- 管理调用网关的业务应用，实现接入认证、授权、额度与监控治理。

## 2. 功能范围
- 应用档案：应用 ID、名称、归属部门、负责人、状态。
- 认证授权：AppKey/AppSecret、IP 白名单、可用模型范围。
- 配额管理：应用级、模型级、供应商级额度设置。
- 频控管理：每秒/每分钟请求限制，支持时段化策略。
- 监控告警：成功率、错误率、耗时、token 消耗。

## 3. 核心业务规则
- AppId 全局唯一，不可复用。
- Secret 明文只展示一次，后续仅支持重置。
- 被禁用应用不能调用任何推理接口。
- 配额可继承部门默认策略并允许应用覆盖。

## 4. 关键页面
- 应用列表页。
- 应用授权页。
- 配额频控配置页。
- 应用调用监控页。

## 5. 关键接口（建议）
- `GET /api/apps`
- `POST /api/apps`
- `PUT /api/apps/{id}`
- `PUT /api/apps/{id}/quota`
- `GET /api/apps/{id}/metrics`

## 6. 指标与验收
- 应用鉴权准确率 100%。
- 应用维度调用统计分钟级可见。
