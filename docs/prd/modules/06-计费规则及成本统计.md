# 计费规则及成本统计模块产品文档

## 1. 模块目标
- 建立精细化计费体系，打通调用行为、成本核算、对账与结算流程。

## 2. 功能范围
- 计费规则：按请求数/token/并发/模型/供应商/应用计费。
- 计费模式：固定单价、阶梯、套餐、自定义规则。
- 成本统计：按时间、应用、模型、供应商、部门聚合分析。
- 成本明细：单次调用账单明细与导出。
- 对账结算：供应商账单对账、内部结算单生成。

## 3. 核心业务规则
- 调用日志与计费记录一一关联。
- 账单周期关闭后只允许冲正，不允许直接修改。
- 规则生效采用版本化（生效时间点切换）。
- 超额调用支持“停止服务”或“超额费率”两种策略。

## 4. 关键页面
- 计费规则管理页。
- 成本分析看板。
- 成本明细页。
- 对账结算页。

## 5. 关键接口（建议）
- `GET /api/billing/rules`
- `POST /api/billing/rules`
- `GET /api/billing/summary`
- `GET /api/billing/details`
- `POST /api/billing/reconciliation`

## 6. 指标与验收
- 账单数据可追溯、可导出、可复算。
- 供应商账单差异可定位到调用明细。
